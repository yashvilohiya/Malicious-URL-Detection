# Malicious-URL-Detection
Implemented Malicious URL detection using a machine learning model that can classify URLs as malicious or benign using data preprocessing, feature extraction, model training, and prediction.

Procedure:
Data Collection: You’ll need a labeled dataset of malicious and benign URLs
Data Preprocessing: Clean and prepare the URL strings. 
Feature Extraction: Extract features that help classify URLs, such as URL length, Number of dots (.), Presence of “@”, “-”, “//”, Number of digits, and use of HTTPS
Model Selection: Choose a classification model.
Model Training and Evaluation
